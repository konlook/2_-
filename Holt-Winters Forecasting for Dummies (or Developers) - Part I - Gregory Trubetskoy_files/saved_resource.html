                                                                                                                                                                    grisha Mod Ondrej 6 years ago edited

                                                                                                                                                                    There isn't a clear consensus on what constitutes the best prediction. One possibility is that it's the last observed value in the series times alpha plus the last smoothed value times 1 - alpha. In Python it should be `result.append(alpha * series[n] + (1 - alpha) * result[n])` just before the return of the `exponential_smoothing()` function (where n is 6 in our specific example). HTH!

                                                                                                                                                                        I modified example from part II (excluding trend parameter) and find out that using only level parameter causes repeating of last smoothed value except of real prediction.                                                                                                                                                                         Here is the code i used:

                                                                                                                                                                        def exponential_smoothing(series, alpha):
                                                                                                                                                                        __result = [series[0]]
                                                                                                                                                                        __for n in range(1, len(series)+1):
                                                                                                                                                                        ____if n == 1:
                                                                                                                                                                        ______level = series[0]
                                                                                                                                                                        ____if n >= len(series): # we are forecasting
                                                                                                                                                                        ______value = result[-1]
                                                                                                                                                                        ____else:
                                                                                                                                                                        ______value = series[n]
                                                                                                                                                                        ____level = alpha*value + (1-alpha)*level
                                                                                                                                                                        ____result.append(level)
                                                                                                                                                                        __return result
